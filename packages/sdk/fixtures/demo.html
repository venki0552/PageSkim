PageSkim SDK demo

This page loads dist/pageskim.min.js from a bare <script> tag — no bundler, no build step — and does three things: renders the badge below, logs a savings report to the console, and exposes the generated sibling in the DOM for agents.

How it works

The badge element converts the live DOM after load and shows raw-HTML tokens versus PageSkim tokens. Token counts use the built-in bytes/4 estimate; uncomment the tokenizer addon script below for o200k-based estimates (it is a couple of megabytes, which is exactly why it is optional).

Try it in the console

PageSkim.savings(document.documentElement.outerHTML)
PageSkim.fromDocument(document).llmMd
